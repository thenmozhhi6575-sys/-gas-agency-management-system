# Gas Agency Management System

A full-stack web application for gas cylinder booking 
and customer management built using PHP and MySQL.

## 🛠 Tech Stack
- **Language:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS

## 👤 User Features
- Register and login securely
- Browse and book gas cylinders online
- Track booking and delivery status

## 🛡️ Admin Features
- Manage all customer details
- View and update booking status
- Control delivery management

## ✨ How It Works
1. Customer registers and logs in
2. Places gas cylinder booking request
3. Admin reviews and updates delivery status
4. Customer tracks booking in real-time
